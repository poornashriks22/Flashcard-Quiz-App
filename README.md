💡 Project Overview:
Developed a Flashcard Quiz Application designed to enhance studying through an interactive and user-friendly experience.

🔧 Key Features:
• Flashcards with a question on the front and answer on the back
• “Show Answer” functionality for better self-assessment
• Smooth navigation using Next and Previous buttons
• Ability to add, edit, and delete flashcards for customization
• Designed with a clean and simple UI for ease of use

🛠 Technologies Used:
HTML | CSS | JavaScript
